4

The Agent Ledger

A transparent audit trail of all economic activity within an organization.

The ledger tracks:

  • Hiring Events: When one agent delegates a subtask to another.
  • Resource Costs: Exact credit deductions for OpenAI/OpenRouter token usage.
  • Top-ups: When credits are added to an organization.
