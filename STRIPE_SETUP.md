## 2. Configure Webhooks
The platform requires a webhook to handle subscription updates and credit grants.

1.  Go to **Stripe Dashboard > Developers > Webhooks**.
2.  Add an endpoint: `https://your-domain.com/api/billing/webhook`.
3.  Select the following events:
    *   `checkout.session.completed` (Crucial for initial payment)
    *   `customer.subscription.updated`
    *   `customer.subscription.deleted`
    *   `invoice.paid` (Optional, for recurring credit grants)
4.  Copy the **Signing Secret** (`whsec_...`) and add it to your Vercel Environment as `STRIPE_WEBHOOK_SECRET`.
